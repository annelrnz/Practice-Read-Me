# Help A Friend: Your Online Bestie

      A program that will motivate the users by displaying motivational and positive quotes to help uplift the user's mood. The program will also enable the user to contribute and help others by prompting them to enter quotes that would be passed and be seen by the next user of the program. It would allow the user to select their current mood to match the quotes that will display.
